Important FAQ

What is the Document Object Model (DOM)?
The DOM is a programming interface for web documents.
It represents the page so that programs can change
the document structure, style, and content.
How do you select an element from the DOM?
You can select an element from the DOM using methods
like getElementById , querySelector, getElementsByClassName
and getElementByTagName.
What is event delegation in the context of the DOM, and why is it useful?
Event delegation is a technique of handling events efficiently
by adding a single event listener to a parent element that can handle all
events of a particular type for its child elements.
